# @mjagency/ui

Shared component library and design system for all 12 MJAgency apps. Contains Tailwind v4 theme tokens in `styles/theme.css` (imported by every app's root layout), reusable React components, and animation primitives. M004 (Design System + Theme Engine) fills this package with the full token set, base components, and the agency-switchable theme engine. At M001 this is a typed stub ensuring all apps can import the package and the workspace typechecks cleanly.
