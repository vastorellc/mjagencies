# @mjagency/config

The foundational configuration package for all MJAgency apps and packages. Exports the shared Pino logger factory (`createLogger`) with a locked redact path list covering passwords, tokens, emails, phone numbers, JWT claims, Stripe/R2/Doppler keys (D-12, REQ-307, REQ-426), plus `edgeLog` for Edge-runtime-safe structured logging (Pino does not work in Next.js middleware — pitfall 3.4). Also exports agency constants (`AGENCIES` array, `AGENCY_PORT_BASE`), the `REDIS_KEY` helper for agency-namespaced Redis/BullMQ prefixes (CLAUDE.md §8), and the ESLint 9 flat config that bans `jsonwebtoken` (REQ-502), `dangerouslyAllowSVG` (SEC-N4), and `NEXT_PUBLIC_*KEY` access (REQ-503). The `tsconfig/base.json` in this package is the base TypeScript config every app and package inherits transitively.
