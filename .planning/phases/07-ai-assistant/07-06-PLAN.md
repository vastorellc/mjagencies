<objective>
XML-wrap user input AND detect jailbreak attempts to prevent prompt injection (REQ-085):

1. `wrapUserInput(text)` wraps in `<user_content>\n${text}\n</user_content>` to separate untrusted content from system instructions
2. `detectJailbreakAttempt(text)` rule-based classifier returning `true` if any of these patterns matches:
   - "ignore previous instructions" / "ignore all instructions" / "ignore prior" (case-insensitive)
   - "you are now" / "act as" / "pretend (you are|to be)"
   - "DAN" mode / "developer mode"
   - System prompt override markers: `system:`, `[SYSTEM]`, `###SYSTEM`, `</user_content>` (closing tag spoofing)
   - Excessive escape sequences (>5 in 100 chars)
3. `guardPrompt(text)` returns `{ safe: false, sanitized: '', reason }` if jailbreak detected; otherwise `{ safe: true, sanitized: wrapUserInput(text) }`
4. Wire into `generate-content.ts`: call BEFORE `redactPii()`. If not safe, throw `PromptInjectionError`.
5. Server actions in `ai-editor.ts`: catch the error and return `{ success: false, error: 'generation-failed', text: 'This input cannot be processed. Please rephrase.' }`

Wave 3 because this depends on the redactPii pipeline established in 07-05 (prompt-guard runs FIRST in the chain).

Output: prompt-guard.ts, tests, generate-content.ts updates, ai-editor.ts catch blocks, index.ts exports.
</objective>

       ```typescript
       /**
        * packages/ai/src/prompt-guard.ts
        * Phase 7 — XML wrapping + jailbreak classifier (REQ-085).
        *
        * Pipeline order in generate-content.ts:
        *   1. guardPrompt(userPrompt)  ← THIS file
        *   2. redactPii(safe.sanitized)
        *   3. fetch LiteLLM
        *
        * Rule-based classifier — matches a curated set of jailbreak signatures.
        * Not exhaustive; layered defense alongside system-prompt instructions and provider safety filters.
        */

       export interface GuardResult {
         safe: boolean
         sanitized: string
         reason?: string
       }

       export class PromptInjectionError extends Error {
         constructor(reason: string) {
           super(reason)
           this.name = 'PromptInjectionError'
         }
       }

       export const JAILBREAK_PATTERNS: RegExp[] = [
         /ignore\s+(?:previous|prior|all|the)\s+(?:instructions?|prompts?|commands?|rules?)/i,
         /(?:you\s+are\s+now|act\s+as|pretend\s+(?:you\s+are|to\s+be))\s+\S+/i,
         /\bDAN\s+(?:mode|persona)\b|\bDAN\b(?=\s|$)/,  // exact 'DAN' standalone — not lowercase 'dan'
         /\bdeveloper\s+mode\b/i,
         /(?:^|\n|\.\s|;\s)system\s*:/i,
         /\[\s*SYSTEM\s*\]/i,
         /###\s*SYSTEM/i,
         /<\/user_content>/i,  // closing-tag spoofing attempt
         /\bjailbreak\b/i,
       ]

       export function wrapUserInput(text: string): string {
         return `<user_content>\n${text}\n</user_content>`
       }

       export function detectJailbreakAttempt(text: string): boolean {
         if (!text) return false

         // Pattern matches
         for (const pattern of JAILBREAK_PATTERNS) {
           if (pattern.test(text)) return true
         }

         // Excessive escape sequences (>5 in any 100-char window)
         const sample = text.slice(0, 1000)  // bound for perf
         const escapes = (sample.match(/\\[ntrbf"'\\]/g) ?? []).length
         if (escapes > 5 && sample.length <= 200) return true

         return false
       }

       export function guardPrompt(text: string): GuardResult {
         if (detectJailbreakAttempt(text)) {
           return {
             safe: false,
             sanitized: '',
             reason: 'Prompt injection attempt detected',
           }
         }
         return {
           safe: true,
           sanitized: wrapUserInput(text),
         }
       }
       ```
