    Create `scripts/gsd-headless.mjs` as a Node ESM script:

    ```javascript
    #!/usr/bin/env node
    /**
     * scripts/gsd-headless.mjs
     * Pre-launch gate: runs all quality checks and exits 0 iff all pass.
     * Usage: node scripts/gsd-headless.mjs [--skip=check1,check2] [--help]
     * Exit codes: 0 = all green, 1 = one or more checks failed.
     *
     * Checks (in order):
     *   1. typecheck   — pnpm tsc --noEmit (all packages)
     *   2. unit-tests  — pnpm turbo run test (Vitest unit + integration)
     *   3. e2e-smoke   — Playwright 5 critical paths × 12 agencies (requires E2E_BASE_URL)
     *   4. axe-wcag    — axe-core WCAG AA scan (requires running app)
     *   5. zap-passive — OWASP ZAP baseline passive scan (requires running app + ZAP installed)
     *   6. lighthouse  — Lighthouse CI (LCP<2.5s, CLS<0.1, FID<100ms) (requires running app)
     *   7. csp-grep    — Static grep for forbidden patterns (NEXT_PUBLIC_.*KEY, jsonwebtoken, dangerouslyAllowSVG)
     */
    ```

    Implementation:
    - Parse args: `--skip=check1,check2` (comma-separated check names to skip), `--help`
    - `const failures = []` accumulator

    For each of the 7 checks, implement as a runCheck(name, fn) wrapper:
    ```javascript
    async function runCheck(name, fn) {
      if (skipped.includes(name)) {
        console.log(`[gsd-headless] SKIP: ${name}`)
        return
      }
      try {
        console.log(`[gsd-headless] RUN: ${name}`)
        await fn()
        console.log(`[gsd-headless] PASS: ${name}`)
      } catch (err) {
        console.error(`[gsd-headless] ERROR: ${name} failed`)
        console.error(`::error::gsd-headless: ${name} failed: ${err.message}`)
        failures.push(name)
      }
    }
    ```

    Check implementations:
    1. `typecheck`: `execSync('pnpm tsc --noEmit', { stdio: 'inherit' })` — throws on non-zero
    2. `unit-tests`: `execSync('pnpm turbo run test', { stdio: 'inherit' })`
    3. `e2e-smoke`: If `E2E_BASE_URL` is not set, log "[gsd-headless] WARN: e2e-smoke skipped (E2E_BASE_URL not set)" and return without failing. Otherwise: `execSync('pnpm turbo run test:e2e', { stdio: 'inherit', env: { ...process.env, E2E_BASE_URL: process.env.E2E_BASE_URL } })`
    4. `axe-wcag`: If `E2E_BASE_URL` is not set, skip with warning. Otherwise: `execSync(`node -e "import('@axe-core/cli').then(({default: axe}) => axe(['${process.env.E2E_BASE_URL}/', '--exit']))"`, { stdio: 'inherit' })`
    5. `zap-passive`: If `ZAP_TARGET_URL` is not set, skip with warning. Otherwise: `execSync(`docker run --rm -t ghcr.io/zaproxy/zaproxy:stable zap-baseline.py -t ${process.env.ZAP_TARGET_URL} -I`, { stdio: 'inherit' })`
    6. `lighthouse`: If `LHCI_SERVER_URL` is not set OR `E2E_BASE_URL` is not set, skip with warning. Otherwise: `execSync('npx @lhci/cli@0.14 autorun', { stdio: 'inherit' })`
    7. `csp-grep`: Run grep synchronously using Node `fs`:
       ```javascript
       import { execSync } from 'node:child_process'
       const forbiddenPatterns = [
         { pattern: 'NEXT_PUBLIC_.*KEY', label: 'NEXT_PUBLIC_ secret key' },
         { pattern: 'jsonwebtoken', label: 'banned jsonwebtoken import' },
         { pattern: 'dangerouslyAllowSVG', label: 'dangerouslyAllowSVG usage' },
       ]
       for (const { pattern, label } of forbiddenPatterns) {
         try {
           const result = execSync(`grep -rn "${pattern}" apps/ packages/ --include="*.ts" --include="*.tsx" --include="*.js" --include="*.mjs" 2>/dev/null || true`, { encoding: 'utf8' })
           if (result.trim().length > 0) {
             throw new Error(`Found ${label}:\n${result.trim()}`)
           }
         } catch (err) {
           if (err.stdout || err.message.includes('Found')) throw err
         }
       }
       ```

    Final exit:
    ```javascript
    if (failures.length > 0) {
      console.error(`[gsd-headless] FAILED: ${failures.length} check(s) failed: ${failures.join(', ')}`)
      process.exit(1)
    }
    console.log('[gsd-headless] ALL CHECKS PASSED — ready for launch.')
    process.exit(0)
    ```

    Add `chmod 755` hint in package.json scripts: `"gsd-headless": "node scripts/gsd-headless.mjs"` in root package.json.

  <action>
    Create `.github/workflows/pre-launch-gate.yml`:

    ```yaml
    name: Pre-Launch Gate
    on:
      workflow_dispatch:
        inputs:
          target_url:
            description: 'Base URL of deployed app for live checks (E2E/axe/ZAP/Lighthouse)'
            required: false
            default: ''
      push:
        branches: [main]

    concurrency:
      group: pre-launch-gate-${{ github.ref }}
      cancel-in-progress: false

    jobs:
      pre-launch-gate:
        name: gsd headless — all checks
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v4
          - uses: pnpm/action-setup@v4
            with:
              version: 10
          - uses: actions/setup-node@v4
            with:
              node-version-file: ".nvmrc"
              cache: "pnpm"
          - uses: dopplerhq/cli-action@v3
          - run: pnpm install --frozen-lockfile

          - name: REQ-501 Payload version pin check
            run: pnpm list payload --depth=0 | grep "payload 3.82.1" || (echo "::error::REQ-501: payload must be 3.82.1 exact" && exit 1)

          - name: REQ-502 No jsonwebtoken usage
            run: |
              COUNT=$(grep -rn "from 'jsonwebtoken'\|require('jsonwebtoken')" apps/ packages/ --include="*.ts" --include="*.tsx" --include="*.js" 2>/dev/null | grep -v node_modules | wc -l)
              if [ "$COUNT" -gt "0" ]; then
                echo "::error::REQ-502: jsonwebtoken found in codebase (use jose only)"
                exit 1
              fi

          - name: Run gsd-headless (static checks)
            run: node scripts/gsd-headless.mjs --skip=e2e-smoke,axe-wcag,zap-passive,lighthouse
            env:
              DOPPLER_TOKEN: ${{ secrets.DOPPLER_CI_TOKEN }}

          - name: Run gsd-headless (live checks — if target_url provided)
            if: ${{ inputs.target_url != '' || github.event_name == 'workflow_dispatch' }}
            run: node scripts/gsd-headless.mjs --skip=typecheck,unit-tests,csp-grep
            env:
              DOPPLER_TOKEN: ${{ secrets.DOPPLER_CI_TOKEN }}
              E2E_BASE_URL: ${{ inputs.target_url }}
              ZAP_TARGET_URL: ${{ inputs.target_url }}
              LHCI_SERVER_URL: ${{ secrets.LHCI_SERVER_URL }}
    ```

    Key decisions:
    - Static checks (typecheck, unit-tests, csp-grep) always run — no live app required
    - Live checks (e2e-smoke, axe-wcag, zap-passive, lighthouse) only run when `target_url` input is provided (workflow_dispatch) or a live URL is available
    - `concurrency: cancel-in-progress: false` prevents a canary deploy from cancelling a running gate
    - REQ-501 and REQ-502 checks are inlined as explicit named steps (per main.yml pattern)
    - `DOPPLER_TOKEN` injected on steps needing env vars
  </action>
