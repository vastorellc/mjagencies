## Tasks

### Task 1: Write root layouts for all 11 agency apps

For each agency app, create `apps/web-{agency}/src/app/layout.tsx`. The template is identical to `apps/web-main/src/app/layout.tsx` (written in 08-01) except for the agency slug passed to `getDataAttrs`.

Agency → slug mapping:
- `web-ecommerce` → `'ecommerce'`
- `web-growth` → `'growth'`
- `web-webdev` → `'webdev'`
- `web-ai` → `'ai'`
- `web-branding` → `'branding'`
- `web-strategy` → `'strategy'`
- `web-finance` → `'finance'`
- `web-engineering` → `'engineering'`
- `web-product` → `'product'`
- `web-video` → `'video'`
- `web-graphic` → `'graphic'`

Template (replace `{SLUG}` with the agency slug):
```typescript
import type { ReactNode } from 'react'
import { getDataAttrs } from '@mjagency/ui'
import './globals.css'

const FOUC_SCRIPT = `(function(){try{var s=localStorage.getItem('mj-theme');var d=window.matchMedia('(prefers-color-scheme: dark)').matches;document.documentElement.setAttribute('data-theme',s||(d?'dark':'light'));}catch(e){}})();`

export default function RootLayout({ children }: { children: ReactNode }): ReactNode {
  const attrs = getDataAttrs({ agency: '{SLUG}' })
  return (
    <html suppressHydrationWarning lang="en" {...attrs}>
      <head>
        <script dangerouslySetInnerHTML={{ __html: FOUC_SCRIPT }} />
      </head>
      <body>{children}</body>
    </html>
  )
}
```

Also create `apps/web-{agency}/src/app/globals.css` for each agency:
```css
@import "@mjagency/ui/styles/theme.css";
```

### Task 2: Write frontend layouts with per-agency niche fonts

For each agency, create `apps/web-{agency}/src/app/(frontend)/layout.tsx` loading the correct `next/font/google` font per the UI-SPEC font table. Each layout exports real `metadata` with non-placeholder title and description, and renders `WebVitalsReporter`.

Per-agency font imports (exact `next/font/google` import names):

| Agency | Font Import | Variable Name |
|--------|------------|---------------|
| ecommerce | `Inter` | `inter` |
| growth | `Plus_Jakarta_Sans` | `plusJakartaSans` |
| webdev | `JetBrains_Mono` (heading) + `Inter` (body) | `jetbrainsMono`, `inter` |
| ai | `Instrument_Sans` | `instrumentSans` |

| finance | `Source_Serif_4` (heading) + `Source_Sans_3` (body) | `sourceSerif4`, `sourceSans3` |
| engineering | `IBM_Plex_Sans` | `ibmPlexSans` |
| product | `DM_Sans` | `dmSans` |
| video | `Bebas_Neue` (heading) + `Inter` (body) | `bebasNeue`, `inter` |
| graphic | `Space_Grotesk` | `spaceGrotesk` |

For agencies with a single font, inject `variable: '--font-brand'`. For agencies with heading + body fonts, inject `variable: '--font-brand'` on the heading font and `variable: '--font-body'` on the body font.

Example for `ecommerce` (single font):
```typescript
import type { Metadata } from 'next'
import type { ReactNode } from 'react'
import { Inter } from 'next/font/google'
import { WebVitalsReporter } from '@mjagency/ui'

const inter = Inter({ subsets: ['latin'], variable: '--font-brand', display: 'swap' })

export const metadata: Metadata = {
  title: 'MJ Ecommerce Agency — Revenue-Driven Online Store Growth',
  description: 'We build and scale ecommerce brands that convert. Shopify, headless, CRO, and paid acquisition — all under one roof.',
}

export default function FrontendLayout({ children }: { children: ReactNode }): ReactNode {
  const ga4Id = process.env['NEXT_PUBLIC_GA4_ID'] ?? ''
  return (
    <>
      <div className={inter.variable}>{children}</div>
      <WebVitalsReporter ga4MeasurementId={ga4Id} />
    </>
  )
}
```
