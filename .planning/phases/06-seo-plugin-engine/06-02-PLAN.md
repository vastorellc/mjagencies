  <name>seo-classic scoring plugin</name>
  <files>packages/seo/src/plugins/seo-classic.ts, packages/seo/src/__tests__/seo-classic.test.ts</files>

  <behavior>
  Seven weighted sub-factors (total 100 pts) per RESEARCH.md Pattern 2:

  1. Title length (20 pts):
     - titleLen >= config.titleMinChars AND <= titleMaxChars → 20 pts
     - titleLen > 0 but outside range → 8 pts (partial)
     - no title → 0 pts

  2. Meta description (15 pts):
     - metaLen >= config.metaDescMinChars AND <= metaDescMaxChars → 15 pts
     - metaLen > 0 but outside range → 5 pts
     - no meta → 0 pts

  3. Focus keyword density (20 pts):
     - density >= keywordDensityMin AND <= keywordDensityMax → 20 pts
     - density > 0 but below min → 10 pts
     - density > keywordDensityMax (stuffing) → 0 pts
     - no focusKeyword or wordCount=0 → 0 pts

  4. Word count (25 pts — graduated):
     - score = Math.round(Math.min(wordCount/wordCountFloor, 1) * 25)

  5. H1 (10 pts):
     - exactly 1 H1 → 10 pts
     - 0 or >1 H1 → 0 pts

  6. H2 (5 pts):
     - at least 1 H2 → 5 pts
     - no H2 → 0 pts

  7. Internal links (5 pts):
     - internalLinks >= internalLinkMin → 5 pts
     - partial: Math.round((internalLinks / internalLinkMin) * 5)

  Returns SeoClassicResult { score: Math.min(100, sum), findings[] }
  Each sub-factor produces a finding: { rule: 'title-length', passed: boolean, detail: 'Title is 45 chars — good' }

  Test cases:
  - Perfect content (all factors met with defaults) → score = 100
  - Empty content (no title, no meta, no words) → score = 0
  - Title = 45 chars, meta = 150 chars, focusKeyword at 1.2%, wordCount = 1500, 1 H1, 2 H2, 3 internal links → score = 100
  - Title = 0 chars → title finding passed=false
  - Title = 30 chars (below min 40) → 8 pts for title (partial credit)
  - wordCount = 750 (half of 1500 floor) → word count contributes Math.round(0.5 * 25) = 12 pts
  - keyword density = 3% (above max 2.5%) → keyword score = 0 pts
  - 0 H1 → H1 finding passed=false, 0 pts
  - 2 H1 → H1 finding passed=false, 0 pts (multiple H1 is SEO error)
  - agencyConfig with titleMinChars=50 overrides default (40) — verifies config is respected
  - findings array length equals 7 (one per sub-factor)
  </behavior>
